African American population percentage in Minnesota by 2010 Census Tract

It is African American (Black) History Month, and I wanted to map some Census data, so this maps shows the percentage of the population that designated themselves as African American, utilizing OpenStreetMap data and US Census Bureau data (via IRE). The basics on how this was accomplished. Fork on Github.

  1. The map was created with TileMill.
  2. Starting with OSM Bright and OSM data for Minnesota.
  3. Utilizing a state boundary from the US Census Bureau, the shape of the Minnesota coule be "cut" out.
  4. IRE provides massaged Census data. Downloaded the CSV of Race and population for Minnesota by Census Tract, as well as the corresponding shapefile.
  5. Imported these two sets into the PostGIS database that was set up for the OSM part.
  6. Then, it was a matter of utilizing Carto within TileMill to create the correct look of the map.
  7. Added some UTF Grid interactivity.
  8. Hosted the tiles on Mapbox.
  9. Finally, with the help of Wax, Modest Maps, and HTML5Boilerplate, this page was created.
